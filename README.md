Данный репозиторий создан для сборки кода, отдельно от отладочного репозитория /DuressKeyboard.

Сокращённое описание приложения:

Это клавиатура для подготовки к эстренным ситуациям. Когда настроена, может стирать данные при вводе специального кода. Работает даже на экране блокировки. Поможет в тех случаях если вас заставляют ввести пароль, тогда вы вводите код сброса вместо пароля. Гораздо надёжнее вариантов которые используют спецвозможности, ведь иногда спецвозможности отключаются системой.
