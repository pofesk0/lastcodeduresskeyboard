Данный репозиторий создан для сборки кода, отдельно от отладочного репозитория /DuressKeyboard.

This repository is created for building code, separately from the debug repository /DuressKeyboard.

Сокращённое описание приложения (Brief description of the application):

Это клавиатура для подготовки к экстренным ситуациям. Когда полностью настроенна, может стирать данные (сбрасывать настройки телефона) при вводе специального кода. Она работает даже на экране блокировки. Это поможет, если кто-то попытается заставить вас ввести пароль. В таком случае вы вводите код сброса вместо пароля и нажимаете стрелку Enter (⏎). 

Вы устанавливаете свой код сброса заранее. 

Это гораздо надежнее, чем приложения, использующие спецвозможности для поиска кода, потому что спецвозможности иногда отключаются системой для сторонних приложений, даже после того как вы их вручную включили.

Крд сброса работает только на экране блокировки, чтобы случайно вы не стерли данные при обычном использовании.

This is keyboard to prepare for emergency situations. When finally configured, it can erase data (reset the phone settings) when you enter a special code. It works even on the lock screen. It help if in the future someone tries to force you to enter a password. In that case, you enter the reset code instead password and press the Enter arrow (⏎). 

You set your own reset code!

This is much more reliable than app's that use accessibility features to find the code, because accessibility features are sometimes disabled by the system for third-party applications, even after you turned them on manually. 

The reset code works only on the lock screen.
